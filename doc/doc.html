Postup inštalácie

Pre inštaláciu sú nutné technológie Spring, jdk17, postgresql a Node.js

Príkazy pre vytvorenie databázy:

  1. CREATE DATABASE pis;
  2. GRANT ALL PRIVILEGES ON DATABASE "pis" TO
  3. GRANT ALL PRIVILEGES ON DATABASE "pis" TO postgres;

Inštalácia frontendu

  1. Prechod do priečinka src/frontend
  2. npm install
  3. npm run build

Spustenie aplikácie

  1. Prechod do priečinka src/frontend
  2. npm run build
  3. Prechod do koreňového priečinka
  4. ./mvnw clean install
  5. Spustenie aplikácie
    • ./mvnw spring-boot:run
  6. Spustenie frontendu
    • npm run serve
